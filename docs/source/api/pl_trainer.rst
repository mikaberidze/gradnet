pl_trainer (module)
===================

Full-featured PyTorch Lightning trainer for ``gradnet``. Requires the optional
``pl`` extra::

    pip install gradnet[pl]

For the lightweight default trainer (no PL dependency), see
:doc:`trainer </api/trainer>`.
